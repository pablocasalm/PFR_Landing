# Padel Film Room Landing

Landing page long-form para captacion de waitlist de **Padel Film Room**.

## Stack

- React + TypeScript + Vite
- TailwindCSS
- Framer Motion

## Variables de entorno

Crea un `.env` a partir de `.env.example` y configura:

```bash
VITE_WAITLIST_WEBHOOK_URL=https://tu-webhook.com/waitlist
```

Si la variable no esta definida, el formulario muestra un error controlado.

## Arranque

```bash
npm install
npm run dev
```
